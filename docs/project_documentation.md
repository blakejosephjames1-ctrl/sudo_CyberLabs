# Project goals
### Primary goals
Design and implement a web-based CyberSecurity training platform to provide education in CyberSecurity and Penetrasting testing. Allowing users to access on demand Virtual Lab enviroments.
### Functional goals
+ Browser-based Lab access
+ Multi-user access with VM isolation
+ Structured learning in 'rooms'
+ realistic attack scenarios
### Technical goals
- use a hypervisor based architecture
- Esure scalable backend
- Secure access control
- maintanable
### Success Criteria
#### For my project to be considered succesfull my inital simple requirments are:
- A user can open a browser, start a room, and access Kali Linux within minutes
- At least five distinct rooms function correctly end-to-end
- Multiple users can use the platform simultaneously without interference
- Labs start, reset, and shut down automatically and reliably
